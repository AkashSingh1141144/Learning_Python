# Object Types / Data Types

- Number : 1234, 3.14, 0xff, 0b1111111
- String : "Hello", 'c'
- Boolean : true, false
- Null : null
- Undefined : undefined
- Symbol : Symbol('hello')
- Object : {}
- List : []
- Tuple : ()
- Set : set('abc'), {'a', 'b', 'c'}
- Map : new Map()
- Function : function(){}
- Class : class{}
- Dictionary : {'food' : 'apple', 'price' : 10}, dict(hours = 10)
- File : open('file.txt', 'r')
- None : None
- Advance: Decorators, Generators, Iterators, MetaProgramming
